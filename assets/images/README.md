# Signature Images

## Cara Menambahkan Tanda Tangan

1. Simpan gambar tanda tangan Budi Hermawan dengan nama: `signature_budi.png`
2. Simpan gambar tanda tangan Iskandar dengan nama: `signature_iskandar.png`
3. Letakkan file-file tersebut di folder ini (`assets/images/`)

## Format Gambar
- Format: PNG (dengan background transparan lebih baik)
- Ukuran rekomendasi: 300x150 pixels
- Warna: Hitam/Abu-abu pada background putih/transparan

## Catatan
Jika file gambar tidak tersedia, sistem akan menampilkan placeholder garis untuk tanda tangan manual.
